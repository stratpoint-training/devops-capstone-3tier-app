DevOps Engineering Course

Course Overview

Hands-on training in modern DevOps practices through project-based learning. You'll build a complete 3-tier application while learning containerization, orchestration, CI/CD, GitOps, and comprehensive observability with Prometheus, Grafana, Loki, and Tempo.

Learn while building: Follow along with the video resources as you implement each component of your project.

Duration: 4 weeks

Local Environment Setup

Choose one of the following for local Kubernetes setup:

Docker Desktop

All-in-one solution with Kubernetes support for Windows/Mac/Linux

Recommended for Easy Setup ✨
Rancher Desktop

Complete container management solution with built-in Kubernetes

Recommended for Production-ready environment ✨
Minikube

Single-node Kubernetes cluster, good for learning

Kind (Kubernetes IN Docker)

Lightweight, uses Docker containers as nodes

k3s

Lightweight Kubernetes for IoT & Edge computing

k3d

Runs k3s clusters in Docker, good for CI/CD environments

💡 Note: We recommend:

  • Docker Desktop for the easiest setup across all operating systems
  • Rancher Desktop if you need a production-ready environment with advanced features

⚠️ Important Container Runtime Note:

If you choose Minikube, k3s, or other options without built-in Docker support, you'll need to separately install a container runtime (like Docker or containerd) before setting up Kubernetes.

  • Docker Desktop: Includes container runtime
  • Rancher Desktop: Includes container runtime
  • Others: Require separate container runtime installation

Observability Stack Implementation

🔍 Prometheus

Metrics collection and monitoring system

  • Scrapes metrics from applications
  • Time-series database
  • Powerful query language (PromQL)
  • Alerting capabilities

📊 Grafana

Visualization and dashboarding platform

  • Beautiful dashboards
  • Multiple data source support
  • Alerting and notifications
  • User management

📝 Loki

Log aggregation system

  • Horizontally scalable
  • Multi-tenant
  • LogQL query language
  • Cost-effective storage

🔗 Tempo

Distributed tracing backend

  • OpenTelemetry support
  • High volume tracing
  • Cost-effective storage
  • Grafana integration

Project Details

Important: Repository Setup

The base code is hosted on GitHub but must be migrated to GitLab for the project:

  1. Clone the GitHub repository: git clone https://github.com/stratpoint-training/devops-capstone-3tier-app.git
  2. Create a new GitLab (gitlab.com) repository
  3. Update the remote origin: git remote set-url origin [your-gitlab-repo-url]
  4. Push to GitLab: git push -u origin main

Base Repository: 3-tier-app Template

Project Focus

Your task is to deploy the 3-tier application located in the /app directory using DevOps practices and tools.

Repository Structure

        /app                    # Your deployment target
        ├── frontend           # React frontend
        ├── backend            # Node.js API
        ├── database          # PostgreSQL setup

        /docs                   # Implementation guides
